How to run:

To run tic tac toe, NOTE this will run a sample game with output and then 
100 games of minimax against qlearning and plot the result 
(can uncomment lines in the main to use other players):

cd tictactoe
python tictactoe.py

To run connect4, NOTE this will run two sample games with output and a plot and then 
100 games of minimax against default player (this will take a few minutes) and plot the result 
(can uncomment lines in the main to use other players):

cd connect4
python connect4.py
